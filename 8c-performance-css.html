3. Performance CSS

Apa itu Performance CSS?

Performance CSS adalah praktik menulis CSS yang:

  • cepat diproses browser
  • tidak memicu reflow/repaint berlebihan
  • mudah di-maintain
  • tidak membebani rendering

‚ùó CSS yang buruk bisa membuat website terasa lambat, meskipun JS dan server cepat.

Bagaimana Browser Memproses CSS? (Ringkas tapi Penting)

  • Browser membaca HTML
  • Membaca CSS ‚Üí membangun CSSOM
  • Menggabungkan DOM + CSSOM ‚Üí Render Tree
  • Layout (reflow) ‚Üí hitung posisi & ukuran
  • Paint
  • Composite
