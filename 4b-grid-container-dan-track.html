1. Grid Container & Grid Track (Rows dan Columns)

CSS Grid selalu dimulai dari grid container.
Tanpa container, tidak ada grid.

    .container { 

        display: grid; 

    }

Begitu ini ditulis:

  • .container menjadi grid container
  • semua anaknya menjadi grid items

a. Grid Track: Kolom dan Baris

Grid dibangun dari:

  • kolom (columns)
  • baris (rows)
Kita mendefinisikannya dengan:

    .container { 

        display: grid; 

        grid-template-columns: 100px 200px 100px; 

        grid-template-rows: 80px 200px; 

    }

Artinya:

  • 3 kolom → 100px | 200px | 100px
  • 2 baris → 80px | 200px
Maka grid terlihat seperti ini:

    +-----+----------+-----+
    |     |          |     |  ← 80px
    +-----+----------+-----+
    |     |          |     |  ← 200px
    +-----+----------+-----+

b. Jumlah kolom menentukan struktur layout

Misalnya layout website:

    grid-template-columns: 200px 1fr;

Artinya:

  • kolom kiri = sidebar
  • kolom kanan = konten
Tanpa HTML rumit, kita sudah punya layout.

c. Satuan khusus Grid: fr

fr berarti fraction of free space

    grid-template-columns: 1fr 2fr 1fr;

Total = 4 bagian
Maka:

  • kolom 1 = 25%
  • kolom 2 = 50%
  • kolom 3 = 25%
Ini lebih fleksibel dari %.

d. Menggabungkan px, %, dan fr

    grid-template-columns: 150px 2fr 1fr;

Artinya:

  • kiri tetap 150px
  • sisa ruang dibagi 3
  • kolom tengah dapat 2 bagian
  • kolom kanan 1 bagian
Inilah kekuatan Grid: campuran fixed dan fleksibel

e. grid-template-rows

Baris bekerja sama seperti kolom:

    grid-template-rows: 80px 1fr 60px;

Contoh layout:

  • header = 80px
  • konten = fleksibel
  • footer = 60px
Tanpa flexbox, tanpa height hack.

f. repeat() untuk efisiensi

Daripada menulis:

    grid-template-columns: 1fr 1fr 1fr 1fr;

Kita bisa:

    grid-template-columns: repeat(4, 1fr);

Ini sangat berguna untuk galeri dan card grid.

g. auto-fit dan auto-fill

Grid bisa membuat kolom otomatis:

    grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));

Artinya:

  • setiap kolom minimal 200px
  • sisanya fleksibel
  • jumlah kolom otomatis sesuai lebar layar
Inilah dasar responsive grid tanpa media query.
