B. Main Axis & Cross Axis (Flexbox)

1. Mengapa Flexbox punya axis?

Flexbox tidak berpikir dalam:

  • kiri – kanan – atas – bawah
Flexbox berpikir dalam:
  • arah aliran dan arah silang
Itulah:
  • Main Axis
  • Cross Axis

2. Apa itu Main Axis?

Main Axis adalah:

  • arah utama aliran flex item
Default: 

    .container { 

        display: flex; 

        flex-direction: row; 

    }

➡ Main axis = horizontal (kiri → kanan)

    [A] [B] [C]

3. Apa itu Cross Axis?

Cross axis adalah:

  • arah yang tegak lurus terhadap main axis
  • Jika main axis horizontal → cross axis vertical

    Main → 

    [A] [B] [C] 

    ↑ 

    Cross

4. Mengubah arah main axis

    .container { 

        display: flex; 

        flex-direction: column; 

    }

Sekarang:

Main axis = atas → bawah

Cross axis = kiri → kanan 

                [A] 

                [B] 

                [C]

5. Kenapa ini sangat penting?

Karena:

Semua properti Flexbox mengacu pada axis, bukan arah layar

6. Contoh Properti

Properti Mengatur
justify-content sepanjang main axis
align-items sepanjang cross axis
align-content cross axis multi-line
flex-grow main axis
flex-shrink main axis
flex-basis main axis

7. Kesalahan umum

Banyak orang mengira:

    justify-content = horizontal 

    align-items = vertical

Itu hanya benar jika flex-direction: row

Kalau: 

    flex-direction: column;

Maka:

    justify-content → vertical 
 


    align-items → horizontal 


                
8. Visualisasi

    flex-direction: row 

    Main axis   → horizontal 

    Cross axis  ↓ vertical 
 


    flex-direction: column 

    Main axis   ↓ vertical 

    Cross axis  → horizontal

9. Analogi

Bayangkan kereta:

  • Rel → main axis
  • Jarak antar rel → cross axis
Kereta bergerak di rel, bukan di udara.

10. Ringkasan

  • Flexbox tidak pakai kiri/kanan
  • Pakai main axis & cross axis
  • flex-direction menentukan arah main axis
  • Semua alignment mengikuti axis
