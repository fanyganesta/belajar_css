Overflow

Overflow adalah mekanisme CSS untuk mengatur apa yang terjadi ketika konten lebih besar dari ruang elemen.

Materi ini erat sekali hubungannya dengan height, width, dan box model.

1. Apa itu overflow?

overflow menentukan:

apa yang dilakukan browser ketika konten keluar dari batas box 

    .box { 

        width: 200px; 

        height: 100px; 

        overflow: hidden; 

    }

üìå Tanpa overflow, browser tetap menggambar konten walau keluar box

2. Jenis-jenis overflow

Nilai Perilaku
visible (default) konten keluar box
hidden konten dipotong
scroll selalu muncul scrollbar
auto scrollbar muncul jika perlu

Dalam materi ini kita fokus ke hidden dan scroll.

3. overflow: hidden

Fungsi

  • menyembunyikan konten yang keluar
  • memotong elemen di luar box

    .card { 

        height: 150px; 

        overflow: hidden; 

    }

Kapan digunakan?

‚úî Crop gambar
‚úî Menyembunyikan teks berlebih
‚úî Membersihkan float (cara lama)
‚úî Animasi slide-in
‚úî Masking elemen

Contoh Praktis 

    .image-box { 

        width: 200px; 

        height: 200px; 

        overflow: hidden; 

    }

‚û° gambar besar akan terpotong rapi

‚ö† Bahaya overflow: hidden

‚ùå Konten penting bisa hilang
‚ùå Aksesibilitas buruk
‚ùå User tidak tahu ada konten terpotong

üìå Gunakan hanya jika disengaja

4. overflow: scroll

Fungsi

  • memaksa scrollbar muncul
  • walau konten cukup

    .box { 

        height: 150px; 

        overflow: scroll; 

    }

‚û° scrollbar selalu terlihat

Kapan digunakan? ‚úî Area chat
‚úî Log aktivitas
‚úî Tabel panjang
‚úî Sidebar list panjang

Kekurangan

‚ùå Scrollbar muncul meski tidak perlu
‚ùå Bisa mengganggu desain

5. overflow: auto (Alternatif Lebih Baik)

    .box { 

        height: 150px; 

        overflow: auto; 

    }

‚û° scrollbar muncul hanya jika diperlukan

üìå Paling sering dipakai di dunia nyata

6. Overflow Arah Spesifik

Kita bisa mengatur sumbu tertentu:

    .box { 

        overflow-x: hidden; 

        overflow-y: auto; 

    }

‚û° horizontal dipotong
‚û° vertical bisa scroll

Contoh Kasus Nyata 

    .table-wrapper { 

        max-width: 100%; 

        overflow-x: auto; 

    } 

    
 
    .table-wrapper { 

        max-width: 100%; 

        overflow-x: auto; 

    }

‚û° tabel besar tetap bisa di-scroll horizontal

7. Hubungan Overflow dengan Height

Overflow tidak bekerja tanpa batasan ruang. 

    .box { 

        overflow: auto; 

    }

‚ùå Tidak akan scroll jika:

tidak ada height / max-height

‚úî Harus ada: 

    .box { 

        max-height: 300px; 

        overflow-y: auto; 

    }

8. Overflow dan Flexbox

Masalah umum:

    .container { 

        display: flex; 

        height: 100vh; 

    } 

        

    .content { 

        overflow: auto; 

    }

‚û° Konten tidak scroll

Solusi 

    .content { 

        overflow: auto; 

        min-height: 0; 

    }

üìå Flex item default-nya min-height: auto

9. Overflow dan Text

Potong teks satu baris

    .text { 

        white-space: nowrap; 

        overflow: hidden; 

        text-overflow: ellipsis; 

    }

‚û° menghasilkan ...

10. Kesalahan Umum Overflow

‚ùå Menggunakan overflow tanpa height
‚ùå Memotong konten penting
‚ùå Scroll di dalam scroll (nested scroll)
‚ùå Lupa aksesibilitas keyboard

11. Best Practice Overflow

‚úî Gunakan auto daripada scroll
‚úî Batasi dengan max-height
‚úî Gunakan scroll hanya pada area tertentu
‚úî Hindari nested scroll
‚úî Perhatikan mobile UX
