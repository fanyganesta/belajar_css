Responsive Typography

Responsive typography adalah:

teks yang menyesuaikan diri dengan ukuran layar dan jarak baca

Bukan cuma mengecil atau membesar — tetapi tetap nyaman dibaca.

Masalah teks tanpa responsive

Di HP:

  • judul kepanjangan
  • baris terlalu rapat
  • paragraf melebar
Di desktop:
  • teks terlalu kecil
  • banyak ruang kosong

Unit yang benar

Hindari:

    font-size: 12px;

Gunakan:

    font-size: rem; 

    em; 

    vw;

Dasar rem

    html { 

        font-size: 16px; 

    } 
 


    p { 

        font-size: 1rem; /* 16px */ 

    } 

    h1 { 

        font-size: 2rem; /* 32px */ 

    }

Jika di mobile kamu ubah:

    html { font-size: 14px; }

Semua teks otomatis menyesuaikan.

Fluid typography

    h1 { 

        font-size: clamp(1.5rem, 4vw, 3rem); 

    }

Artinya:

  • minimal 1.5rem
  • ideal mengikuti lebar layar
  • maksimal 3rem

Tanpa media query.

Line length & line height

Teks nyaman jika:

  • 50–75 karakter per baris
  • line-height ≈ 1.4 – 1.7

 
    p { 

        max-width: 60ch; 

        line-height: 1.6; 

    }

Mobile first typography

    body { font-size: 14px; } 
 


    @media (min-width: 768px) { 

        body { font-size: 16px; } 

    } 
 


    @media (min-width: 1200px) { 

        body { font-size: 18px; } 

    }

Kesalahan umum

❌ pakai px untuk semua teks
❌ judul terlalu besar di HP
❌ baris terlalu panjang di desktop
