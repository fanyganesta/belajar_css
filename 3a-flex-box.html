CSS Satu Arah ‚Äì Flexbox

1. Mengapa Flexbox Ada?

Sebelum Flexbox, layout CSS dibuat dengan:

  • float
  • inline-block
  • position
  • table

Semua itu bukan dibuat untuk layout, tapi dipaksa untuk layout.
Akibatnya:

  • susah sejajar
  • vertikal center ribet
  • responsive berantakan
  • perlu banyak hack
Flexbox diciptakan untuk satu tujuan:

Mengatur tata letak elemen dalam satu arah secara fleksibel dan otomatis.

2. Apa maksud ‚ÄúCSS satu arah‚Äù?

Flexbox bekerja dalam satu dimensi:

  • horizontal atau
  • vertikal
  • Tidak bisa dua arah sekaligus.
üìå Itulah sebabnya disebut:

Layout satu arah (one-dimensional layout system)

Kalau mau dua arah ‚Üí CSS Grid.

3. Konsep Dasar Flexbox

Flexbox selalu terdiri dari:

  • Flex Container
  • Flex Items

    .container { 

        display: flex; 

    }

Saat sebuah elemen diberi display: flex, maka:

  • dia menjadi flex container
  • semua anak langsungnya menjadi flex items

HTML: 
    <div class="container">
    <div>A</div>
    <div>B</div>
    <div>C</div>
    </div>
CSS: 
    .container { 

        display: flex; 

    }

‚û° A, B, C otomatis sejajar dalam satu baris

4. Apa yang membuat Flexbox ‚Äúpintar‚Äù?

Flexbox tidak bekerja seperti:

‚Äúsetiap elemen berdiri sendiri‚Äù

Tetapi:

codsatu sistem layout yang mengatur semua anak sekaligus

Flexbox menghitung:

  • ukuran item
  • ruang kosong
  • arah
  • jarak
  • distribusi
  • dan menyusunnya secara otomatis

5. Filosofi Flexbox

Flexbox bukan tentang:

‚Äúdi mana posisi elemen ini?‚Äù

tetapi:

‚Äúbagaimana ruang dibagi ke elemen-elemen ini?‚Äù

Ini membuat:

  • center otomatis
  • distribusi merata
  • responsif tanpa media query
  • sejajar tanpa kalkulasi manual

6. Perbedaan dengan width & height biasa

Tanpa Flexbox:

    .box { 

        width: 33%; 

    }

Jika satu box lebih besar ‚Üí layout rusak.

Dengan Flexbox:

    .container { 

        display: flex; 

    }

Browser:

  • membagi ruang
  • menyesuaikan
  • merapikan secara otomatis

7. Kapan Flexbox dipakai?

Flexbox cocok untuk:

  • navbar
  • sidebar + content
  • card list
  • form layout
  • button group
  • dashboard
  • header/footer
Tidak cocok untuk:
    layout dua arah kompleks (pakai Grid)

8. Cara berpikir dengan Flexbox

Saat memakai Flexbox, kita tidak berpikir:

‚ÄúElemen ini di kiri 200px‚Äù

Tetapi:

‚ÄúBagaimana elemen-elemen ini dibagi di dalam container?‚Äù

Itu sebabnya properti Flexbox selalu berbicara tentang:

  • alignment
  • distribusi
  • arah
  • proporsi
  • bukan koordinat

9. Gambaran besar

Flexbox itu:

  • sistem tata letak berbasis aliran dan ruang
  • Bukan posisi mutlak.

10. Ringkasan

Flexbox adalah:

  • sistem layout satu arah
  • berbasis container
  • otomatis mengatur anak-anaknya
  • fokus pada pembagian ruang
  • dibuat khusus untuk UI modern

Flex Container & Flex Item

1. Apa itu Flex Container?

Flex container adalah elemen induk yang mengaktifkan sistem Flexbox.

Dia dibuat dengan:

    .container {  

        display: flex; 

    }
Begitu sebuah elemen menjadi display: flex, maka:
  • dia menjadi flex container
  • semua anak langsungnya menjadi flex item

Contoh:

    <div class="container"> 

    <div class="box">A</div> 

    <div class="box">B</div> 

    <div class="box">C</div> 

    </div>

                
    .container { 

        display: flex; 

    }

‚û° A, B, C otomatis masuk sistem Flexbox

2. Siapa yang termasuk Flex Item?

Hanya anak langsung dari flex container.

    <div class="container"> 

    <div>A</div>       <!-- flex item --> 

    <div> 

        <span>B</span> <!-- B BUKAN flex item --> 

    </div> 

    </div>

üìå span bukan flex item karena bukan anak langsung.

3. Tugas Flex Container

Flex container bertanggung jawab atas:

  • arah layout
  • alignment
  • jarak
  • wrapping
  • distribusi ruang
Ia mengatur semua anak sekaligus.

4. Tugas Flex Item

Flex item bertanggung jawab atas:

  • ukuran relatif
  • urutan
  • kemampuan tumbuh atau menyusut
  • basis ukuran
Flex item tidak mengatur posisi ‚Äî container yang mengatur.

5. Perbedaan Cara Kerja

Flex Container Flex Item
Mengatur arah Mengatur ukuran
Mengatur alignment Mengatur proporsi
Mengatur jarak Mengatur urutan
Mengatur wrapping Mengatur pertumbuhan

6. Contoh Interaksi

    .container { 

        display: flex; 

    } 
 


    .item1 { flex: 1; } 

    .item2 { flex: 2; } 

    .item3 { flex: 1; }

‚û° Container membagi ruang
‚û° Item menentukan proporsinya

7. Tanpa Flex Container, Flex Item tidak berfungsi

    .box { 

        flex: 1; 

    }

‚ùå Tidak akan bekerja kalau parent bukan display: flex

Flexbox selalu sistem dua pihak:

  • container
  • item

8. Analogi Dunia Nyata

Bayangkan rak buku:

  • Rak = flex container
  • Buku = flex item
Rak menentukan:
  • arah susunan
  • jarak antar buku
  • apakah baris baru
Buku menentukan:
  • tebal
  • seberapa banyak ruang diambil

9. Kesalahan Umum

‚ùå Memberi flex ke elemen tanpa parent display:flex
‚ùå Mengira cucu elemen ikut flex
‚ùå Mengatur alignment di item bukan di container
