Grid Lines, Grid Cell, dan Grid Area

Kalau sebelumnya kita membuat peta grid (rows & columns), sekarang kita belajar:

bagaimana item ditempatkan di dalam peta itu

Grid Line (Garis Grid)

Setiap kolom dan baris dipisahkan oleh garis.

Contoh:

    grid-template-columns: 1fr 1fr 1fr;

Ini menghasilkan:

    | 1 | 2 | 3 | 4 |

Ada 4 garis, bukan 3.
Garis diberi nomor otomatis:

    1   2   3   4
    |---|---|---|

Begitu juga dengan baris:

    grid-template-rows: 100px 200px;

Maka garisnya:

    1
    | 100px
    2
    | 200px
    3

Grid Cell

Grid cell adalah kotak terkecil hasil perpotongan 1 baris dan 1 kolom.

    +-----+-----+
    |  A  |  B  |  ← masing-masing = 1 grid cell
    +-----+-----+
    |  C  |  D  |
    +-----+-----+

Setiap elemen default akan menempati 1 cell.

Grid Area

Grid area adalah kumpulan beberapa cell.

Misalnya kita ingin elemen menempati:

  • dari kolom 1 sampai 3
  • dari baris 1 sampai 2

    .item {  

        grid-column: 1 / 3; 

        grid-row: 1 / 2; 

    }

Artinya:

dari garis 1 ke garis 3

Cara kerja grid-column dan grid-row

    .item { 

        grid-column: 2 / 4; 

        grid-row: 1 / 3; 

    }

Berarti:

mulai di garis kolom 2

berakhir di garis kolom 4
→ melebar 2 kolom

mulai di baris 1

berakhir di baris 3
→ setinggi 2 baris

Span (lebih praktis)

Daripada menghitung garis:

    grid-column: 2 / span 2;

Artinya:

mulai dari kolom 2, lebar 2 kolom

grid-area shorthand

    .item { 

        grid-area: 1 / 2 / 3 / 4; 

    }

Artinya:

row-start / column-start / row-end / column-end

Kenapa ini penting?

Flexbox mengatur alur
Grid mengatur posisi absolut dalam layout

Dengan grid line:

  • kita bisa bikin layout majalah
  • dashboard
  • web admin
  • layout tidak simetris
Tanpa nested div.

Contoh nyata layout

    .container { 

        display: grid; 

        grid-template-columns: 200px 1fr; 

        grid-template-rows: 60px 1fr 50px; 

    } 
 


    .header { grid-column: 1 / 3; } 

    .sidebar { grid-row: 2 / 3; } 

    .content { grid-column: 2 / 3; } 

    .footer { grid-column: 1 / 3; }

Ini membuat layout website profesional.
