Align Items

1. Apa itu align-items?

align-items adalah properti untuk:

mengatur posisi flex item di sepanjang cross axis

Kalau justify-content mengatur arah utama,
align-items mengatur arah sampingnya.

2. Ingat Axis-nya

flex-direction Main Axis Cross Axis
row horizontal vertical
column vertical horizontal

align-items selalu bekerja di cross axis

3. Nilai align-items

a. stretch (default)

align-items: stretch;

Item akan memanjang memenuhi cross axis

Jika row:

tinggi item mengikuti tinggi container

    ||A| |B| |C|      |    
    || | | | | |      |
    || | | | | |      |

b. flex-start

align-items: flex-start;

Item berada di awal cross axis

    
  ||A||B||C|       |
  |                |
  |                |

c. flex-end

align-items: flex-end;

Item berada di akhir cross axis

    
    |                |
    |                |
    ||A||B||C|       |

d. center

align-items: center;

Item berada di tengah cross axis

    
    |                |
    ||A||B||C|       |
    |                |
(vertikal center jika row)

e. baseline

align-items: baseline;

Item disejajarkan berdasarkan teks di dalamnya

Dipakai untuk:

  • form
  • label
  • typography

4. Contoh Praktis

Vertical center

    .container { 

        display: flex; 

        align-items: center; 

    }

Jika row:
➡ semua item vertikal tengah

Horizontal center (column)

    .container { 

        display: flex; 

        flex-direction: column; 

        align-items: center; 

    }

5. Perbedaan dengan justify-content

Properti Mengatur
justify-content main axis
align-items cross axis

6. Kesalahan umum

❌ Menggunakan align-items untuk horizontal padahal row
❌ Lupa flex-direction
❌ Mengira align-items = vertical align selalu
