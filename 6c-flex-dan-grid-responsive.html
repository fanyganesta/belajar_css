Flex & Grid Responsive

Flexbox dan Grid bukan hanya alat layout — mereka adalah mesin adaptasi untuk responsive design.

Tanpa Flex & Grid, responsive itu sulit dan penuh hack.

Kenapa float & inline-block tidak cukup?

Dulu orang membuat responsive dengan:

  • float
  • inline-block
  • clear
  • width %
Masalahnya:
  • tidak fleksibel
  • susah sejajar
  • susah berubah bentuk
Flex & Grid diciptakan untuk:

mengatur ruang secara dinamis

Flexbox untuk satu arah

Flexbox ideal untuk:

  • navbar
  • list item
  • card horizontal
  • toolbar
Flex bersifat:

satu dimensi (row atau column)

Contoh Flex Responsive

    .menu { 

        display: flex; 

        flex-direction: column; /* mobile */ 

        gap: 10px; 

    } 
 


    @media (min-width: 768px) { 

        .menu { 

            flex-direction: row; /* desktop */ 

        } 

    }

HP → menu vertikal
Desktop → menu horizontal

Grid untuk dua arah

Grid ideal untuk:

  • galeri
  • dashboard
  • card product
  • layout halaman
Grid bersifat:

dua dimensi (baris & kolom)

Contoh Grid Responsive

    .cards { 

        display: grid; 

        grid-template-columns: 1fr; /* mobile */ 

        gap: 20px; 

    } 
 


    @media (min-width: 768px) { 

        .cards { 

            grid-template-columns: 1fr 1fr; 

        } 

    } 
 


    @media (min-width: 1200px) { 

        .cards { 

            grid-template-columns: 1fr 1fr 1fr; 

        } 

    }

Tanpa JS, layout berubah sendiri.

auto-fit & minmax (power feature)

    .cards { 

        display: grid; 

        grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); 

    }

Artinya:

Buat sebanyak mungkin kolom, minimal 250px, maksimal fleksibel

Ini sering menghilangkan kebutuhan breakpoint.

Flex wrapping

    .items { 

        display: flex; 

        flex-wrap: wrap; 

    }

Item otomatis turun baris saat ruang habis.

Perbedaan peran

Flex Grid
Baris / kolom Baris + kolom
Komponen Layout halaman
Sederhana Struktur besar

Dalam responsive:

  • Grid → struktur
  • Flex → isi
