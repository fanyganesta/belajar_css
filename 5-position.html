Ukuran & Posisi Besar

A. position: absolute

1. Pengertian position: absolute
position: absolute adalah properti CSS yang digunakan untuk mengatur posisi elemen secara bebas terhadap elemen induk tertentu (bukan alur normal halaman).
Elemen dengan position: absolute:

  • Keluar dari alur normal dokumen
  • Tidak memengaruhi posisi elemen lain
  • Diposisikan menggunakan properti:
    • top
    • right
    • bottom
    • left
Contoh sederhana: 

.box { 

    position: absolute; 

    top: 50px; 

    left: 100px; 

}

2. Cara Kerja position: absolute

Elemen dengan position: absolute akan mencari elemen induk terdekat yang memiliki properti: 

    position: relative | absolute | fixed | sticky

Jika tidak ditemukan, maka elemen akan:

  • Diposisikan relatif terhadap <body>

Contoh Struktur Dasar

    <div class="container"> 

        <div class="box"></div> 

    </div>

    .container { 

        position: relative; 

        width: 300px; 

        height: 300px; 

        background: lightgray; 

    } 


    .box { 

        position: absolute; 

        width: 100px; 

        height: 100px; 

        background: blue; 

        top: 20px; 

        right: 20px; 

    }

üìå Kesimpulan: .box diposisikan relatif terhadap .container, bukan halaman.

3. Absolute untuk Elemen Berukuran Besar

position: absolute sering digunakan untuk:

  • Background dekoratif
  • Overlay
  • Banner
  • Badge / label
  • Elemen visual besar yang tidak mengganggu layout utama
Contoh Elemen Dekorasi Besar 

.decor-bg { 

    position: absolute; 

    width: 600px; 

    height: 600px; 

    background: rgba(0, 123, 255, 0.1); 

    top: -200px; 

    right: -200px; 

    border-radius: 50%; 

}

Efek:

  • Lingkaran besar
  • Keluar dari kontainer
  • Tidak memengaruhi teks atau elemen lain

4. Kombinasi Ukuran & Posisi

Properti yang sering digunakan:

  • width / height
  • max-width
  • top, left, right, bottom
  • transform
Contoh posisi tengah: 

.center-box { 

    position: absolute;  

    top: 50%;  

    left: 50%;  

    transform: translate(-50%, -50%);  

}

üìå Teknik ini sering dipakai untuk:

  • Modal
  • Popup
  • Hero text

6. Penggunaan z-index pada Absolute

Karena elemen absolute sering tumpang tindih, z-index digunakan untuk mengatur lapisan. 

    .box-1 { 

        position: absolute; 

        z-index: 1; 

    } 


    .box-2 { 

        position: absolute; 

        z-index: 2; 

    }

Semakin besar nilai z-index, semakin di atas posisinya.

7. Kesalahan Umum (Yang Harus Dihindari)

‚ùå Lupa memberi position: relative pada parent
‚ùå Menggunakan absolute untuk seluruh layout utama
‚ùå Ukuran terlalu besar tanpa kontrol overflow
‚ùå Tidak memperhatikan responsive design

8. Best Practice Penggunaan Absolute

‚úî Gunakan hanya untuk elemen dekoratif
‚úî Tetapkan parent sebagai relative
‚úî Kombinasikan dengan transform untuk positioning presisi
‚úî Gunakan media query untuk responsive
‚úî Jangan gunakan untuk konten utama

position: relative

1. Pengertian position: relative

position: relative adalah properti CSS yang digunakan untuk menggeser posisi elemen dari posisi aslinya, tanpa mengeluarkannya dari alur normal halaman.
Artinya:

  • Elemen tetap menempati ruang aslinya
  • Elemen lain tidak ikut bergeser
  • Posisi baru ditentukan dengan:
    • top
    • right
    • bottom
    • left
Contoh: 

    .box { 

        position: relative; 

        top: 20px; 

        left: 30px; 

    }

2. Cara Kerja position: relative

Ketika sebuah elemen diberi position: relative:

  • Browser menggambar elemen di posisi normal
  • Lalu menggeser tampilannya sesuai nilai top, left, dll
  • Ruang awal elemen tetap dipertahankan
üìå Ini berbeda dengan absolute yang menghilangkan elemen dari alur dokumen.

3. relative untuk Elemen Berukuran Besar

position: relative sering digunakan pada elemen besar untuk:

  • Section (hero, banner, card besar)
  • Container dekorasi
  • Pembungkus elemen absolute
Contoh: 

    .section { 

        position: relative; 

        width: 100%; 

        height: 400px; 

        background: #f5f5f5; 

    }

Elemen ini:

  • Tetap menjaga struktur layout
  • Bisa menjadi referensi posisi bagi elemen dekoratif di dalamnya

4. relative sebagai Induk Dekorasi

Fungsi paling penting position: relative bukan untuk menggeser dirinya, tapi untuk:

  • Menjadi acuan posisi bagi elemen absolute di dalamnya
Contoh Kombinasi: 

    <div class="card">
        <span class="badge">NEW</span>
    </div>

.card { 

    position: relative; 

    width: 300px; 

    height: 200px; 

    background: white; 

} 


.badge { 

    position: absolute; 

    top: 10px; 

    right: 10px; 

}

üìå Tanpa relative, .badge akan menempel ke .

5. Penggunaan Ukuran Besar dengan relative

Properti yang sering dikombinasikan:

  • width, height
  • max-width
  • padding
  • overflow
  • z-index
Contoh: 

.decor-section { 

    position: relative; 

    width: 100%; 

    min-height: 500px; 

    overflow: hidden; 

}

Digunakan untuk:

  • Menyimpan dekorasi besar agar tidak keluar layar
  • Mengontrol efek visual

6. z-index pada Elemen Relative

Elemen dengan position: relative bisa menggunakan z-index. 

    .layer-1 { 

        position: relative; 

        z-index: 1; 

    } 


    .layer-2 { 

        position: relative; 

        z-index: 2; 

    }

üìå Sangat berguna untuk:

  • Menumpuk dekorasi besar
  • Mengatur layer UI
