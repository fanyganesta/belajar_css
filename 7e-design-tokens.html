5. Design Tokens (Warna, Spasi, Ukuran)

Design Tokens adalah nilai desain inti yang dijadikan sumber kebenaran tunggal (single source of truth) untuk seluruh UI.

Kalau CSS Variable adalah alatnya,
maka Design Tokens adalah konsep & isinya.

Apa itu Design Tokens?

Design Tokens adalah:

representasi nilai desain (warna, spasi, font, ukuran, radius, dll) dalam bentuk variabel yang dapat digunakan ulang.

Contoh token:

  • Warna utama
  • Jarak standar
  • Ukuran teks
  • Border radius
  • Shadow

Kenapa Design Tokens penting?

Tanpa token:

  • warna tidak konsisten
  • spacing acak
  • perubahan besar = edit banyak file
Dengan token:
  • satu perubahan → seluruh UI ikut berubah
  • mudah kolaborasi designer & developer
  • siap untuk dark mode & rebrand

Jenis-Jenis Design Tokens

1. Color Tokens

    :root { 

        --color-primary: #2563eb; 

        --color-secondary: #16a34a; 

        --color-danger: #dc2626; 
 


        --color-text: #111827; 

        --color-muted: #6b7280; 

        --color-bg: #ffffff; 

    }

Penggunaan:

    .button-primary {    

        background: var(--color-primary); 

        color: white; 

    }

2. Spacing Tokens

Gunakan skala, bukan angka random.

    :root { 

        --space-xs: 4px; 

        --space-sm: 8px; 

        --space-md: 16px; 

        --space-lg: 24px; 

        --space-xl: 32px; 

    } 
 


    .card { 

        padding: var(--space-md); 

        margin-bottom: var(--space-lg); 

    }

3. Typography Tokens

    :root { 

    --font-base: 16px; 

    --font-sm: 14px; 

    --font-lg: 18px; 

    --line-base: 1.6;
    }

4. Radius Tokens

    :root { 

        --radius-sm: 4px; 

        --radius-md: 8px; 

        --radius-lg: 12px; 

    }

5. Shadow Tokens

    :root { 

        --shadow-sm: 0 1px 2px rgba(0,0,0,.05); 

        --shadow-md: 0 4px 10px rgba(0,0,0,.1); 

    }

Token vs Component

❌ Salah:

    .card { 

        padding: 17px; 

        border-radius: 9px; 

    }

✔ Benar:

    .card { 

        padding: var(--space-md); 

        border-radius: var(--radius-md); 

    }

Global Token → Semantic Token

Tingkat lanjut: pisahkan token teknis & makna

    :root { 

        /* Global */ 

        --blue-500: #2563eb; 

        --red-500: #dc2626; 
 


        /* Semantic */ 

        --color-primary: var(--blue-500); 

        --color-error: var(--red-500); 

    }

Keuntungan:

  • rebrand cepat
  • dark mode mudah

Contoh Dark Mode

    [data-theme="dark"] { 

        --color-bg: #111827; 

        --color-text: #f9fafb; 

    }

Struktur File Design System (Best Practice)

    styles/ 

    ├─ tokens.css 

    ├─ base.css 

    ├─ layout.css 

    ├─ components/ 

    │  └─ button.css 

    └─ utilities.css

Kesalahan Umum Design Tokens

❌ Terlalu banyak token
❌ Nama tidak konsisten
❌ Campur token & komponen
❌ Hardcode warna di komponen
