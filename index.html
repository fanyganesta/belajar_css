Berikut penjelasan yang bisa kamu gunakan:

1. CSS Bekerja dengan Cara Memilih Elemen HTML

CSS tidak berdiri sendiri. Ia mencari elemen HTML yang ingin dihias melalui selector.

Contoh:

    p{
        color: blue;
    }

Artinya:

  • CSS mencari semua
  • Lalu menerapkan warna teks biru
üëâ Tanpa HTML, CSS tidak punya objek untuk dihias.

2. Browser Membaca CSS dari Atas ke Bawah

Saat halaman dibuka:

  • Browser memuat HTML
  • Browser memuat CSS
  • CSS dibaca berurutan dari atas ke bawah
Jika ada aturan yang saling bertabrakan, aturan terakhir bisa menimpa aturan sebelumnya, tergantung prioritasnya.

3. CSS Menggunakan Sistem Prioritas (Cascade)

Inilah asal nama Cascading Style Sheets. Urutan prioritas secara sederhana:

  1. !important
  2. Inline CSS 
    (style="")
  3. Internal CSS 
    <style> ... </style>
  4. External CSS 
    (.css) 

     atau 

     <link rel="stylesheet" href="ini-file-css.css" >
  5. Urutan penulisan (yang terakhir menang)
Contoh: 

    p {
    color: red;
    }

    p {
    color: green;
    }

4. CSS Menghitung Spesifisitas Selector

Jika dua aturan sama-sama aktif, browser menghitung siapa yang lebih spesifik. Urutan spesifisitas:

  1. #id
  2. .class
  3. tag
Contoh: 

    p { color: blue; } 

    .text { color: green; } 

    #judul { color: red;}

5. CSS Mewariskan Gaya (Inheritance)

Beberapa properti CSS diturunkan ke anak elemen
Contoh: 

    body {

        font-family: arial;

        color: black;

    }

Semua teks di dalam <body> otomatis:

  • Menggunakan Arial
  • Berwarna hitam
üëâ Tidak semua properti diwariskan (misalnya margin, padding).

6. CSS Diterapkan ke Layout dan Tampilan

Setelah aturan dipilih dan diprioritaskan, browser:

  • Menghitung ukuran (box model)
  • Menentukan posisi (layout)
  • Menggambar tampilan di layar (render)
