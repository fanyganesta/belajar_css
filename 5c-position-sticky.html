position: sticky

position: sticky adalah gabungan antara:

  • relative
  • dan fixed
Ia berperilaku seperti relative, sampai pada titik tertentu… lalu berubah menjadi fixed.

Masalah yang ingin diselesaikan

Bayangkan:

  • kamu punya judul tabel
  • atau sidebar
  • atau menu filter
Kamu ingin:
be Saat halaman di-scroll, elemen itu ikut naik… tapi ketika sudah menyentuh batas atas layar, dia “menempel” di situ.
  • Itu bukan fixed, bukan juga relative.
  • Itulah sticky.

Contoh dasar

HTML:

    <div class="container"> 

    <div class="header">Saya sticky</div> 

    <p>konten panjang…</p> 

    </div>

CSS:

    .header { 

        position: sticky; 
 
        top: 0; 

        background: white; 

    }

Artinya:

Selama belum menyentuh top: 0, dia ikut scroll
Setelah menyentuh, dia terkunci di atas

Perbedaan dengan fixed

fixed sticky
Keluar dari layout Tetap di layout
Menempel ke viewport Menempel di dalam parent
Selalu terlihat Hanya aktif saat di-scroll
  • Sticky tidak keluar dari kontainernya.
  • Jika parent-nya habis, sticky ikut berhenti.

Syarat agar sticky bekerja

Ini sering membuat orang bingung.

1️⃣ Harus ada offset

    top: 0; /* atau bottom, left, right */

2️⃣ Parent tidak boleh punya:

    overflow: hidden; 

    overflow: auto; 

    overflow: scroll;

Karena sticky menempel ke scroll container.

Sticky di dalam container

    .sidebar { 
 
        height: 100vh; 

        overflow-y: auto; 

    } 
 


    .menu { 

        position: sticky; 

        top: 0; 

    }

Sticky akan menempel di dalam sidebar, bukan seluruh halaman.

Use case nyata

Sticky dipakai di:

  • header tabel Excel
  • filter di e-commerce
  • judul kategori
  • sidebar blog
  • TOC (table of contents)

Mental model

Fixed:

“tempel di layar”

Sticky:

“tempel di dalam area ini, tapi hanya kalau sudah sampai batasnya”
