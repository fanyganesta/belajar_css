A. CSS sebagai Dekorasi

1. Cara CSS Bekerja pada Color dan Background

a. Peran Color dan Background dalam CSS

Dalam CSS, color dan background adalah properti dasar yang berfungsi untuk:

  • Memberi identitas visual
  • Meningkatkan keterbacaan teks
  • Menciptakan suasana dan fokus pada elemen tertentu
CSS bekerja dengan cara mencari elemen HTML lalu menerapkan warna dan latar belakang sesuai aturan yang ditulis.

2. Cara CSS Bekerja pada Properti color

Properti color digunakan untuk mengatur warna teks pada elemen HTML.

a. Contoh dasar: 

    p {

        color: blue;

    }

Cara kerjanya:

  • Browser mencari semua elemen <p>
  • CSS menerapkan warna teks biru
  • Teks dalam <p> berubah warna
b. Pewarisan warna (Inheritance) Properti color bersifat diturunkan. 

    body {

    color: #333;

    }

➡️ Semua teks di dalam akan berwarna abu gelap, kecuali ditimpa aturan lain.

3. Cara CSS Bekerja pada Properti background

Properti background digunakan untuk mengatur latar belakang elemen.

a. background-color

Memberi warna latar belakang elemen. 

    div {

    background-color: lightblue;

    }

Cara kerja:

  • CSS hanya memberi warna pada area elemen
  • Tidak diwariskan ke anak elemen

b. background-image

Menampilkan gambar sebagai latar belakang. 

        body {

        background-image: url("bg.jpg");

        }

CSS akan:

  • Memuat gambar
  • Mengulang gambar secara default
  • Menempatkannya dari kiri atas

c. background-repeat

Mengatur pengulangan background. 

    background-repeat: no-repeat;

d. background-position

Menentukan posisi background. 

    background-position: center;

e. background-size

Mengatur ukuran background. 

    background-size: cover;

4. Hubungan Color dan Background

CSS bekerja dengan memperhatikan kontras antara color dan background.
Contoh: 

    .card {

    background-color: #222;

    color: white;

    }

➡️ Background gelap + teks terang = teks mudah dibaca

5. Prioritas CSS pada Color dan Background

Jika satu elemen memiliki lebih dari satu aturan: 

    p {

    color: red;

    }


    .text {

    color: blue;

    }

HTML:

    <p class="text" >Contoh teks</p>

➡️ Warna teks menjadi biru karena .class lebih spesifik.

6. Contoh Kasus Sederhana

    body {

    background-color: #f4f4f4;

    color: #333;

    }


    .box {

    background-color: white;

    color: black;

    }

Cara kerja:

  • body mengatur warna global
  • .box menimpa warna background dan teks
  • CSS menerapkan dekorasi berdasarkan prioritas

7. Kesalahan Umum dalam Penggunaan Color dan Background

❌ Warna teks terlalu mirip background
❌ Mengandalkan background image tanpa background-color cadangan
❌ Terlalu banyak warna dalam satu halaman

2. Cara CSS Bekerja pada Font Size dan Font Family

a. Peran Font dalam Dekorasi Web

Font merupakan elemen penting dalam CSS karena berpengaruh langsung pada:

  • Keterbacaan teks
  • Kenyamanan pengguna
  • Kesan visual (formal, santai, modern, klasik)
CSS bekerja dengan cara mengatur ukuran dan jenis huruf pada elemen HTML melalui properti font-size dan font-family.

b. Cara CSS Bekerja pada font-size

Properti font-size digunakan untuk mengatur ukuran teks.
Contoh dasar: 

    p {

    font-size: 16px;

    }

Cara kerjanya:

  • Browser menemukan elemen <p>
  • CSS menetapkan ukuran teks 16 piksel
  • Teks ditampilkan sesuai ukuran tersebut

i. Satuan font-size
CSS menyediakan beberapa satuan ukuran:

Satuan Keterangan
px Ukuran tetap (paling umum untuk pemula)
% Relatif terhadap elemen induk
em Relatif terhadap ukuran font induk
rem Relatif terhadap font root (html)
Contoh: 
    body {

        font-size: 16px;

    } 


    h1 {

        font-size: 2em; /* 32px */ 

    }

ii. Pewarisan font-size font-size bersifat diturunkan (inheritance). 
    body { 

        font-size: 14px; 

    }
➡️ Semua teks di dalam akan berukuran 14px kecuali diubah.

c. Cara CSS Bekerja pada font-family

Properti font-family digunakan untuk mengatur jenis huruf.
Contoh dasar: 

    body { 

        font-family: Arial; 

    }
Cara kerjanya:
  • Browser mengecek apakah font tersedia di perangkat
  • Jika tersedia → digunakan
  • Jika tidak → pindah ke font cadangan
i. Font Family dengan Cadangan (Fallback) 
    p { 

        font-family: "Segoe UI", Arial, sans-serif; 

    }
Urutan kerja:
  • Gunakan Segoe UI
  • Jika tidak ada → Arial
  • Jika tidak ada → sans-serif
ii. Jenis Font Family Umum
Jenis Contoh
Serif Times New Roman
Sans-serif Arial, Verdana
Monospace Courier New
Cursive Comic Sans MS
Fantasy Impact

d. Hubungan Font Size dan Font Family

CSS bekerja mengombinasikan kedua properti ini untuk menghasilkan teks yang nyaman dibaca.
Contoh: 

    .article { 

        font-family: "Poppins", sans-serif; 

        font-size: 18px; 

    }
➡️ Teks tampil:
  • Ukuran cukup besar
  • Jenis huruf modern dan rapi

e. Prioritas CSS pada Font

Jika ada beberapa aturan font: 

    p {

        font-size: 14px; 

    } 


    .text { 

        font-size: 18px; 

    }
HTML: 
    <p class="text">Contoh teks</p>
➡️ Teks berukuran 18px karena selector class lebih spesifik.

f. Contoh Kasus Sederhana

    body { 

        font-family: Arial, sans-serif; 

        font-size: 16px; 

    } 


    h1 { 

        font-size: 32px; 

    } 


    p { 

        font-size: 16px; 

    }
Cara kerja:
  • body menentukan font global
  • h1 memperbesar ukuran judul
  • p menggunakan ukuran standar

g. Kesalahan Umum Penggunaan Font

❌ Menggunakan terlalu banyak jenis font
❌ Ukuran font terlalu kecil
❌ Tidak menyiapkan font cadangan
❌ Menggunakan px saja tanpa mempertimbangkan responsivitas

h. Aturan Penulisan (Syntax)

  • Gunakan Tanda Kutip (" "): Untuk nama font yang lebih dari satu kata (Contoh: "Open Sans", "Times New Roman").
  • Tanpa Tanda Kutip: Untuk satu kata (Contoh: Arial, Verdana) dan wajib tanpa kutip untuk jenis font generic (Contoh: sans-serif, serif, monospace).
  • Pemisah Koma: Gunakan koma untuk memisahkan daftar font.

Cara CSS Bekerja pada Text Align

a. Pengertian Text Align dalam CSS

text-align adalah properti CSS yang digunakan untuk mengatur perataan teks secara horizontal di dalam suatu elemen. Text align berperan penting dalam dekorasi karena mempengaruhi:

  • Kerapian tampilan
  • Fokus pembaca
  • Struktur visual halaman

b. Cara CSS Bekerja pada text-align

CSS bekerja dengan cara:

  • Mencari elemen HTML yang dituju
  • Mengatur posisi teks di dalam area elemen
  • Meratakan teks sesuai nilai yang diberikan
Contoh: 
    p { 

        text-align: center; 

    }
➡️ Semua teks di dalam elemen <p> akan berada di tengah.

c. Nilai-nilai text-align

Berikut nilai yang umum digunakan:

Nilai Fungsi
left Teks rata kiri (default)
right Teks rata kanan
center Teks rata tengah
justify Teks rata kiri dan kanan

Contoh Penggunaan: 
  
    .left { 

        text-align: left; 

    } 


    .center { 

        text-align: center; 

    } 


    .right { 

        text-align: right; 

    } 


    .justify { 

        text-align: justify; 

    }

d. Area Kerja text-align

text-align hanya bekerja pada teks dan inline content di dalam elemen, bukan memindahkan elemen itu sendiri. Contoh yang benar: 

    div { 

        text-align: center; 

    }
➡️ Teks di dalam <div> menjadi rata tengah
❌ <div>-nya sendiri tidak berpindah posisi

e. Pewarisan text-align

Properti text-align bersifat diturunkan (inheritance). 

    .section { 

        text-align: center; 

    }

➡️ Semua teks di dalam .section akan rata tengah, kecuali ditimpa aturan lain.

f. Prioritas CSS pada Text Align

Jika satu elemen memiliki lebih dari satu aturan: 

    p {

        text-align: left; 

    } 


    .text { 

        text-align: right; 

    }

HTML: 

    <p class="text">Contoh teks</p>

➡️ Teks menjadi rata kanan karena selector class lebih spesifik.

g. Contoh Kasus Sederhana

body {
  text-align: center;
}

.article {
  text-align: justify;
}

Cara kerja:

  • Seluruh teks default rata tengah
  • Teks dalam .article diratakan kiri-kanan karena aturan lebih spesifik

h. Kesalahan Umum Penggunaan Text Align

❌ Menggunakan text-align untuk memusatkan <div>
❌ Terlalu sering menggunakan justify sehingga sulit dibaca
❌ Menggabungkan text align tanpa memahami prioritas CSS
