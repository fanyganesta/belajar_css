CSS Grid Layout

1. Apa itu CSS Grid?

CSS Grid adalah sistem layout dua dimensi.

Artinya:

CSS Grid bisa mengatur baris (row) dan kolom (column) secara bersamaan.

Berbeda dengan Flexbox yang satu arah (hanya baris atau kolom), Grid bekerja seperti kertas kotak-kotak (spreadsheet / tabel).

Kalau Flexbox itu seperti:

menyusun buku dalam satu rak

Maka Grid itu seperti:

menyusun buku di rak dan lantai sekaligus

2. Masalah layout sebelum Grid

Sebelum Grid, kita harus pakai:

  • float
  • inline-block
  • flexbox bersarang
  • framework (Bootstrap)
Layout kompleks (dashboard, galeri, halaman berita) jadi:
  • sulit
  • banyak hack
  • tidak stabil
Grid diciptakan untuk:
  • layout halaman besar dan sistematis

3. Konsep utama CSS Grid

CSS Grid terdiri dari dua bagian:
Komponen Fungsi
Grid Container Elemen induk yang mengaktifkan grid
Grid Items Anak-anak yang ditempatkan di dalam grid 
    .container {  

        display: grid; 

    }

Begitu ini ditulis, anak-anaknya langsung jadi grid items.

4. Perbedaan Grid vs Flexbox

Flexbox Grid
Satu arah Dua arah
Cocok navbar dan list Cocok layout halaman
Konten menentukan ukuran Grid menentukan struktur
Aliran fleksibel Struktur eksplisit

Grid lebih cocok untuk:

  • dashboard
  • layout web
  • galeri
  • halaman majalah
Flexbox lebih cocok untuk:
  • navbar
  • tombol
  • card row

5. Grid bekerja dengan garis (grid lines)

CSS Grid tidak bekerja dengan kotak, tetapi dengan garis. Misalnya 3 kolom berarti ada 4 garis:

    | 1 | 2 | 3 |
    1   2   3   4

Kita menempatkan elemen dengan: dari garis ke garis

Sub Materi CSS Grid

Setelah memahami konsep besarnya, Grid dibagi ke beberapa bagian:

1. Grid Container

operti utama:

  • display: grid
  • grid-template-columns
  • grid-template-rows
  • gap
Ini menentukan bentuk peta grid.

2. Grid Track (Rows & Columns)

Kita mendefinisikan ukuran kolom dan baris:

    .container { 

        grid-template-columns: 200px 1fr 200px; 

        grid-template-rows: auto 1fr auto; 

    }

Artinya:

  • kiri 200px
  • tengah fleksibel
  • kanan 200px

3. Grid Units

Grid punya satuan khusus:

  • fr
  • auto
  • %
  • minmax()
Ini membuat layout responsif tanpa media query.

4. Grid Items

Setiap anak bisa:

  • menempati beberapa kolom
  • melompat baris
  • tumpang tindih
Properti:
  • grid-column
  • grid-row
  • grid-area

5. Auto-placement

Grid bisa:

menempatkan item secara otomatis ke kotak kosong

Ini membuat galeri & dashboard jadi mudah.

6. Grid vs Flexbox bersama

Di web modern:

Grid → struktur besar

Flexbox → isi di dalamny
