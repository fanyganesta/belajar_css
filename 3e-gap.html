Gap (Flexbox)

1. Apa itu gap?

gap adalah properti untuk:
  • mengatur jarak antar flex item secara otomatis
Tanpa perlu:
  • margin
  • kalkulasi
  • reset di item terakhir
    .container { 

        display: flex; 

        gap: 20px; 

    }

➡ Semua item akan terpisah 20px

2. Kenapa gap lebih baik dari margin?

Dengan margin:

    .item { 

        margin-right: 20px; 

    }

Masalah:

  • item terakhir ikut margin
  • perlu : last-child
  • rusak saat wrapping

Dengan gap:

    gap: 20px;
Browser yang mengatur jarak secara cerdas.

3. Cara kerja gap

Flexbox akan:

  • menghitung jarak antar item
  • tidak memberi jarak di pinggir container
  • bekerja otomatis pada wrap

4. Bentuk gap

Satu nilai 

    gap: 20px;

➡ horizontal dan vertical

Dua nilai

    gap: 10px 30px;
Nilai Arah
10px vertical
30px horizontal

5. row-gap dan column-gap

    row-gap: 10px; 

    column-gap: 30px;

Dipakai jika:

ingin jarak beda antara baris dan kolom

6. Gap dan Flex-Wrap

Gap bekerja sempurna dengan wrapping:

    .container { 

        display: flex; 

        flex-wrap: wrap; 

        gap: 20px; 

    }

➡ Jarak antar baris dan kolom rapi tanpa margin hack

7. Gap vs Justify-content

gap = jarak antar item

justify-content = distribusi sisa ruang

Jangan tertukar.

8. Kapan pakai gap?

✔ Card grid
✔ Button group
✔ Navbar item
✔ Icon list
✔ Form layout

9. Kesalahan umum

❌ Pakai margin padahal sudah ada gap
❌ Menggabungkan gap + margin ganda
❌ Mengira gap mengatur pinggir container
