Justify Content

1. Apa itu justify-content?

justify-content adalah properti untuk:

  • mengatur posisi dan distribusi flex item di sepanjang main axis
Ingat:
  • Main axis = arah utama flex (row atau column)

2. Cara kerja sederhananya

Jika item lebih kecil dari container, maka:

  • akan ada sisa ruang

justify-content menentukan:

  • sisa ruang itu ditaruh di mana dan dibagi bagaimana

3. Contoh dasar

    .container { 

        display: flex; 

        justify-content: center; 

    }

Jika ada ruang kosong:

  • semua item dipusatkan di tengah main axis

4. Jenis-jenis justify-content

a. flex-start (default)

    justify-content: flex-start;

Item menumpuk di awal main axis

    [A][B][C]______

b. flex-end

    justify-content: flex-end;

Item menumpuk di akhir main axis

    ______[A][B][C]

c. center

    justify-content: center;

Item berada di tengah

    ___[A][B][C]___

d. space-between

    justify-content: space-between;

Item paling kiri & kanan nempel, sisanya dibagi

    [A]____[B]____[C]

e. space-around

    justify-content: space-around;

Ada ruang di kiri dan kanan tiap item

    _[A]__[B]__[C]_

f. space-evenly

    justify-content: space-evenly;

Semua jarak sama

    _[A]__[B]__[C]_

(tengah dan pinggir sama)

5. Hubungan dengan flex-direction

Jika:

    flex-direction: row; 



    justify-content = horizontal

Jika:

    flex-direction: column; 
 


    justify-content = vertical

6. Kapan justify-content bekerja?

Hanya jika:

  • ukuran total item < ukuran container

Jika item sudah penuh:

  • tidak ada ruang untuk dibagi
  • justify-content tidak terlihat efeknya

7. Contoh Kasus Nyata

Navbar

    nav { 

        display: flex; 

        justify-content: space-between; 

    }

Logo di kiri, menu di kanan.

Center box 

    .container { 

        display: flex; 

        justify-content: center; 

    }

8. Kesalahan umum

❌ Menggunakan justify-content untuk vertical align padahal main axis horizontal
❌ Mengira justify-content selalu horizontal
❌ Tidak paham hubungan dengan flex-direction
