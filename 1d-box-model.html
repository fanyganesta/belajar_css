Box Model

A. Margin

1. Pengertian CSS Box Model

Box Model adalah konsep dasar CSS yang menjelaskan bahwa setiap elemen HTML dianggap sebagai sebuah kotak (box).
Kotak ini terdiri dari beberapa lapisan yang mengatur:

  • ukuran elemen
  • jarak antar elemen
  • tampilan visual (dekorasi)
  • üì¶ Setiap elemen HTML = 1 box

2. Struktur Box Model

Urutan Box Model dari dalam ke luar:

  • Content
  • Padding
  • Border
  • Margin
Visualisasi sederhana: 

    +-----------------------+
    |         Margin        |
    |  +-----------------+  |
    |  |      Border     |  |
    |  |  +-----------+  |  |
    |  |  |  Padding  |  |  |
    |  |  | +-------+ |  |  |
    |  |  | |Content| |  |  |
    |  |  | +-------+ |  |  |
    |  |  +-----------+  |  |
    |  +-----------------+  |
    +-----------------------+

3. Fungsi Box Model dalam Dekorasi

Box Model digunakan untuk:

  • mengatur jarak antar elemen
  • membuat layout rapi
  • memberi ruang napas (whitespace)
  • menciptakan estetika desain
Tanpa box model ‚Üí tampilan sempit dan berantakan.

4. Fokus Materi: Margin

4.1 Pengertian Margin
Margin adalah jarak di luar border, digunakan untuk:

  • memisahkan satu elemen dengan elemen lain
  • mengatur posisi elemen dalam layout

    .box { 

        margin: 20px; 

    }

üìå Margin tidak memiliki warna
üìå Margin berada paling luar pada box model

5. Jenis‚ÄìJenis Margin

5.1 Margin Satu Nilai 

    .box { 

        margin: 20px; 

    }

‚û° Semua sisi (atas, kanan, bawah, kiri) = 20px

5.2 Margin Dua Nilai 

    .box { 

        margin: 10px 20px; 

    }
  • atas & bawah: 10px
  • kiri & kanan: 20px

5.3 Margin Empat Nilai 

    .box { 

        margin: 5px 10px 15px 20px; 

    }

Urutan:
‚û° atas ‚Üí kanan ‚Üí bawah ‚Üí kiri

5.4 Margin Per Sisi 

    .box { 

        margin-top: 10px; 

        margin-right: 15px; 

        margin-bottom: 20px; 

        margin-left: 5px; 

    }

6. Margin dan Dekorasi Layout

Margin sering digunakan untuk:

  • memberi jarak antar card
  • memisahkan section
  • membuat layout lebih lega
Contoh kartu dekoratif: 

    .card { 

        margin: 20px; 

        border: 1px solid #ccc; 

    }

7. Margin Auto (Pusat Elemen)

    .box { 

        width: 300px; 

        margin: 0 auto; 

    }

üìå Elemen otomatis berada di tengah horizontal
üìå Sangat sering digunakan pada container

8. Margin Negatif

Margin bisa bernilai negatif. 

    .box { 

        margin-top: -20px; 

    }

üìå Elemen bisa saling menumpuk
üìå Cocok untuk dekorasi khusus
‚ö† Gunakan dengan hati-hati

9. Margin Collapsing (Penting!)

Margin atas dan bawah bisa saling menyatu (collapse).
Contoh: 

    .box1 { 

        margin-bottom: 30px; 

    } 

 

    .box2 { 

        margin-top: 20px; 

    }

‚û° Jarak antar box = 30px, bukan 50px
üìå Hanya terjadi pada margin vertikal

10. Margin vs Padding (Perbandingan Singkat)

Aspek Margin Padding
Letak Di luar border Di dalam border
Memengaruhi jarak antar elemen ‚úî ‚ùå
Bisa negatif ‚úî ‚ùå
Mempengaruhi ukuran elemen ‚ùå ‚úî

11. Best Practice Penggunaan Margin

‚úî Gunakan margin untuk jarak antar elemen
‚úî Gunakan konsisten (misal: kelipatan 8px)
‚úî Hindari margin negatif berlebihan
‚úî Jangan campur margin & padding tanpa tujuan
‚úî Gunakan margin-bottom untuk vertikal spacing

12. Studi Kasus Sederhana

HTML: 

    <div class="box">Box 1</div> 

    <div class="box">Box 2</div>

CSS: 

    .box { 

        margin-bottom: 20px; 

        padding: 15px; 

        border: 1px solid black;
    }

üìå Box tidak menempel
üìå Tampilan lebih rapi
üìå Mudah dikembangkan

B. Padding

1. Pengertian Padding

Padding adalah jarak di dalam elemen, tepatnya jarak antara content dan border.
üìå Padding memberi ruang napas pada isi elemen
üìå Padding memengaruhi ukuran total elemen
Contoh sederhana: 

    .box { 

        padding: 20px; 

    }

2. Posisi Padding dalam Box Model

Urutan Box Model:
Content ‚Üí Padding ‚Üí Border ‚Üí Margin
Visualisasi: 

    +-----------------------+
    |         Margin        |
    |  +-----------------+  |
    |  |      Border     |  |
    |  |  +-----------+  |  |
    |  |  |  Padding  |  |  |
    |  |  | +-------+ |  |  |
    |  |  | |Content| |  |  |
    |  |  | +-------+ |  |  |
    |  |  +-----------+  |  |
    |  +-----------------+  |
    +-----------------------+

Padding selalu berada di dalam border.

3. Fungsi Padding dalam Dekorasi

Padding digunakan untuk:

  • membuat teks tidak menempel ke border
  • memperindah tombol (button)
  • memperbesar area klik
  • meningkatkan kenyamanan visual (UX)

Tanpa padding:
‚ùå tampilan sempit
‚ùå sulit dibaca

4. Cara Menulis Padding

4.1 Padding Satu Nilai

    .box { 

        padding: 15px; 

    }

‚û° Semua sisi: 15px

4.2 Padding Dua Nilai 

    .box { 

        padding: 10px 20px; 

    }

  • atas & bawah: 10px
  • kiri & kanan: 20px


4.3 Padding Empat Nilai 

    .box { 

        padding: 5px 10px 15px 20px; 

    }

Urutan:
‚û° atas ‚Üí kanan ‚Üí bawah ‚Üí kiri

4.4 Padding Per Sisi 

    .box { 

        padding-top: 10px; 

        padding-right: 15px; 

        padding-bottom: 20px; 

        padding-left: 5px; 

    }

5. Padding dan Ukuran Elemen

Padding menambah ukuran elemen secara default.
Contoh: 

    .box { 

        width: 200px; 

        padding: 20px; 

    }

‚û° Lebar total = 200 + 20 + 20 = 240px
üìå Ini sering menjadi sumber kebingungan pemula.

6. Padding vs Margin (Perbedaan Penting)

Aspek Padding Margin
Letak Di dalam border Di luar border
Mempengaruhi ukuran elemen ‚úî ‚ùå
Bisa negatif ‚ùå ‚úî
Area background ikut ‚úî ‚ùå
Digunakan untuk Ruang isi Jarak antar elemen

7. Padding dan Background

Background ikut mengisi area padding. 

    .box { 

        padding: 20px; 

        background-color: lightblue; 

    }

‚û° Warna background terlihat sampai padding
‚û° Padding membuat elemen tampak lebih besar

8. Padding pada Elemen Umum

8.1 Button

    button { 

        padding: 10px 20px; 

    }

üìå Area klik lebih besar
üìå Tampilan lebih elegan

8.2 Card / Box 

    .card { 

        padding: 20px; 

        border: 1px solid #ccc; 

    }

üìå Konten tidak menempel ke border
üìå Cocok untuk dashboard & blog

9. Padding dalam Layout Responsif

Gunakan padding untuk:

  • jarak aman pada mobile
  • kenyamanan baca
Contoh: 

    .container { 

        padding: 16px; 

    }

üìå Lebih fleksibel daripada margin untuk isi

10. Box-Sizing dan Padding (Sekilas)

Default: 

    box-sizing: content-box;

Alternatif lebih aman: 

    box-sizing: border-box;

Dengan border-box:

  • padding tidak menambah ukuran elemen
  • layout lebih stabil

    .box { 

        width: 300px; 

        padding: 20px; 

        box-sizing: border-box; 

    }

11. Best Practice Penggunaan Padding

‚úî Gunakan padding untuk isi elemen
‚úî Gunakan nilai konsisten (8px, 16px, 24px)
‚úî Gunakan border-box untuk layout kompleks
‚úî Jangan menggantikan fungsi margin dengan padding
‚úî Perhatikan ukuran total elemen

12. Studi Kasus Sederhana

HTML: 

    <div class="card">
        <h3>Judul</h3>
        <p>Ini adalah konten card.</p>
    </div>

CSS: 

    .card { 

        padding: 20px; 

        border: 1px solid black; 

    }

üìå Tampilan rapi
üìå Mudah dikembangkan
üìå Nyaman dibaca

B. Border

1. Pengertian Border

Border adalah garis pembatas yang mengelilingi elemen, terletak di antara padding dan margin.
Urutan Box Model:
Content ‚Üí Padding ‚Üí Border ‚Üí Margin

üìå Border digunakan untuk:

  • memberi batas visual
  • memperjelas struktur elemen
  • memperindah tampilan (dekorasi)
  • membantu debugging layout

2. Posisi Border dalam Box Model

Visualisasi: 

    +-----------------------+
    |         Margin        |
    |  +-----------------+  |
    |  |      Border     |  |
    |  |  +-----------+  |  |
    |  |  |  Padding  |  |  |
    |  |  | +-------+ |  |  |
    |  |  | |Content| |  |  |
    |  |  | +-------+ |  |  |
    |  |  +-----------+  |  |
    |  +-----------------+  |
    +-----------------------+

‚û° Border membungkus padding dan content
‚û° Background elemen tidak menembus border

3. Struktur Properti Border

Border memiliki 3 komponen utama: 

    border: width style color;

Contoh: 

    .box { 

        border: 2px solid black; 

    }
Komponen Fungsi
width ketebalan garis
style jenis garis
color warna border

4. Border Width (Ketebalan)

Contoh: 

    .box { 

        border-width: 1px; 

    }

Nilai yang umum:

  • 1px
  • 2px
  • thin, medium, thick
  • border-width: 2px;

5. Border Style (Jenis Garis)

Jenis border yang sering digunakan:

Style Keterangan
solid garis lurus
dashed garis putus-putus
dotted titik-titik
double garis ganda
groove efek cekung
ridge efek timbul
inset masuk ke dalam
outset keluar

Contoh: 

    .box { 

        border: 2px dashed red; 

    }

üìå Tanpa border-style, border tidak akan muncul.

6. Border Color (Warna)

Contoh: 

    .box {  

        border-color: blue; 

    }

Bisa menggunakan:

  • nama warna (red)
  • hex (#333)
  • rgb / rgba
  • hsl

7. Border per Sisi

Border dapat diatur per sisi: 

    .box { 

        border-top: 2px solid red;  

        border-right: 2px solid green; 

        border-bottom: 2px solid blue; 

        border-left: 2px solid black; 

    }

üìå Berguna untuk:

  • underline custom
  • separator
  • efek hover

8. Border Radius (Sudut Membulat)

Digunakan untuk membuat sudut membulat: 

    .box { 

        border-radius: 10px; 

    }

Variasi: 

    border-radius: 50%; /* lingkaran */ 

    border-radius: 8px 12px;

üìå Sangat umum untuk:

  • card
  • button
  • input form

9. Border dan Ukuran Elemen

Border menambah ukuran elemen (default). 

    .box { 

        width: 200px; 

        border: 5px solid black; 

    }

‚û° Lebar total = 210px
üìå Sama seperti padding, border memengaruhi ukuran elemen.

10. Border dan box-sizing

Dengan border-box: 

    .box { 

        box-sizing: border-box; 

    }

‚úî Border tidak menambah ukuran elemen
‚úî Layout lebih mudah dikontrol

11. Border sebagai Dekorasi UI

11.1 Card 

    .card { 

        border: 1px solid #ddd; 

        padding: 16px; 

        border-radius: 8px; 

    }

üìå Tampilan rapi & profesional

11.2 Button 

    button { 

        border: 2px solid #052B62; 

        padding: 8px 16px; 

        background: transparent; 

    }

üìå Cocok untuk outline button

11.3 Input Form 

    input { 

        border: 1px solid #ccc; 

    }

Focus effect: 

    input:focus { 

        border-color: blue; 

    }

12. Border untuk Debugging

Border sering dipakai untuk melihat struktur layout: 

    * { 

        border: 1px solid red; 

    }

üìå Sangat membantu memahami box model

13. Border vs Outline

Border Outline

Bisa radius ‚úî

14. Best Practice Border

‚úî Gunakan border tipis (1px)
‚úî Hindari terlalu banyak warna
‚úî Gunakan border-radius konsisten
‚úî Kombinasikan dengan padding
‚úî Gunakan outline untuk fokus, border untuk struktur
